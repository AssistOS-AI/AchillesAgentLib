# Assign Distribution Region (Samples)

Expose a long list of distribution centres without overwhelming the user. The assistant samples up
to ten entries per prompt but accepts values across the entire catalogue.

## Required Input

- Distribution region code

## Behaviour

- Samples enumerator options when presenting prompts.
- Accepts user-provided values even if not shown in the sampled list.
- Confirms with the human-readable label and executes with the technical ID.

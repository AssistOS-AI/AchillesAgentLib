# Schedule Project Rotation

Coordinate engineering rotations by collecting timing details and leadership assignments, while
allowing users to revise earlier inputs during confirmation.

## Required Inputs

- Project code
- Location
- Start date
- End date
- Supervisor

## Optional Inputs

- Backup supervisor
- Priority level

## Behaviour

- Supports revising existing values mid-conversation.
- Summarises updated details prior to execution.
- Executes once the user confirms the latest summary.
